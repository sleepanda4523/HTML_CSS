# CSS 포지셔닝          
## CSS 포지셔닝과 주요 속성들       
### CSS 포지셔닝이란?       
평범한 웹 페이지를 CSS를 이용해 꾸미고, CSS 포지셔닝을 이용해 여러 요소를 적절히 배치한다.          
즉 브라우저 화면 안에 각 콘텐츠 영역을 어떻게 배치할지 결정하는 것.     
### box-sizing 속성 -  박스 너비 기준 정하기        
```box-sizing : content-box | border-box```             
앞에서 배운 width와 height 속성은 박스 모델 안 콘텐츠 부분의 너비.      
= 패딩이나 테두리는 따로 계산해야함.
-> width 값을 콘텐츠 영역 너비 값으로 사용할 지, 패딩이나 테두리를 포함시켜 사용할 지 결정.     
#### 속성 값        
+ **content-box** : width 속성 값을 콘텐츠 영역 너비 값으로 사용. 기본값.       
+ border-box : width 속성 값을 콘텐츠 영역에 테두리까지 포함한 박스 모델 전체 너비 값으로 사용.     
### float 속성 - 왼쪽이나 오른쪽으로 배치하기       
```float: left | right | none```            
웹 요소를 어느 방향으로 배치함.     
#### 속성 값     
+ left : 해당 요소를 문서의 왼쪽으로 배치.      
+ right : 해당 요소를 문서의 오른쪽으로 배치.       
+ none : 좌우 어느쪽으로도 배치하지 않음.       
### clear 속성 - float 속성 해제        
```clear: left | right | none | both```     
float 속성을 이용해 웹 페이지 요소를 왼쪽 또는 오른쪽으로 배치하면 그 다음에 넣는 다른 요소들에게도 똑같은 속성이 전달된다. 
-> float 속성이 필요 없음을 알려주는 속성.      
float: left - clear: left가 기본 형식이지만 float가 left인지 right인지 상관없이 무조건 기본 상태로 되돌리고 싶을 때 clear: borh라고 함.     
### position 속성 - 배치방법 지정하기           
```position: static | relative | absolute | fixed```        
웹 문서 안의 요소들을 자유자재로 배치해주는 속성.       
#### 속성 값        
+ static : 요소를 문서의 흐름에 맞추어 배치.        
+ relative : 이전 요소에 자연스럽게 연결해 배치하되 위치를 지정할 수 있다.          
+ absolute : 원하는 위치를 지정해 배치.         
+ fixed : 지정한 위치에 고정해 배치. 화면에서 잘릴 수 있음.     
static을 제외한 나머지는 좌표를 이용해 각 요소의 위치를 조절.       
top, bottom, left, right로 지정. 각각 얼마나 떨어졌는가를 기준.     
### visibility 속성 - 요소를 보이게 하거나 보이지 않게 하기         
```visibility: visible | hidden | collapse```           
특정 요소를 화면에 보이게 하거나 숨기거나 겹치게 설정하는 속성.     
#### 속성 값        
+ **visible** : 화면에 요소를 표시. 기본값.     
+ hidden : 화면에서 요소를 감춤. 다만 크기는 그대로 유지.       
+ collapse : 표의 행, 열, 행/열 그룹에서 지정하면 서로 겹치도록 조정. 그 외 영역에서는 hidden처럼 처리.     
### z-index 속성 - 요소 쌓는 순서 정하기        
```z-index: <숫자>```           
요소 위에 요소를 쌓을 때 쌓는 순서를 지정하는 속성.     
속성 값이 작을 수록 아래에 쌓임.            
## 다단으로 편집하기            
신문처럼 텍스트 단(column)을 다단으로 편집할 수 있다.       
### column-width - 단의 너비 고정하고 다단 구성하기         
```column-width: <크기> | auto```           
단의 너비를 고정해놓고 구성하는 속성.       
화면이 커지면 단의 개수가 많아지고 화면이 작아지면 단의 개수가 줄어듬.  
#### 속성 값        
+ <크기> : 단 너비를 직접 지정.     
+ auto : 단의 개수 같은 다른 속성에 따라 단의 너비가 자동 계산.     
### column-count - 단의 개수 고정하고 다단 구성하기     
```column-count: <숫자> | auto```           
단의 개수를 고정해놓고 구성하는 속성.       
창의 너비가 커지면 단의 너비도 커진다.      
#### 속성 값            
+ <숫자> : 콘텐츠가 들어갈 단의 개수를 지정.        
+ auto : 단의 너비 같은 다른 속성에 따라 단의 개수가 자동 계산.         
### column-gap - 단과 단 사이 여백 조정     
```column-gap: <크기> | normal```           
단과 단 사이 여백을 지정하는 속성.      
#### 속성 값            
+ <크기> : 단과 단 사이 여백을 숫자로 지정.     
+ normal : 여백을 자동으로 지정. W3C에서 권장하는 여백은 1em.       
### column-rule - 구분선의 색상, 스타일, 너비 지정          
```
column-rule-color: <색상>           
column-rule-style: none | hidden | dotted | dashed | solid | double
                    groove | ridge | inset | outset         
column-rule-width: <크기> | thin | medium | thick       
column-rule: <너비> | <색상> | <스타일>
```            
색상, 형태, 너비를 따로 지정할 수도 있고 같이 지정할 수도 있다.     
column-rule-style 속성 값과 border-style 속성 값과 같다.        
### break-after - 다단 위치 지정하기            
```
break-after: column | avoid-column          
break-before: column | avoid-column         
break-inside: column | avoid-column         
```         
이 속성은 단을 나눌 때 위치를 지정하는 속성들.      
특정 요소의 앞, 뒤, 안에서 단을 나누게 됨.      
#### 속성 값        
+ column : 단을 나눔.           
+ avoid-column : 단을 나누지 않음.      
### column-span - 여러단을 하나로 합치기            
```column-span: 1 | all```          
단을 합쳐 표시하는 속성.        
#### 속성 값        
+ **1** : 단을 합치치 않는 것. 기본값.      
+ all : 전체 단을 하나로 합침. 일부만 합치는 건 불가능.     
## 표 스타일            
표의 크기, 테두리, 셀 테두리, 여러가지 여백 등의 표 스타일 구성.        
### caption-side 속성 - 표 제목 위치 정하기     
```caption-side: top | bottom```            
표 제목은 기본적으로 표의 위쪽에 표시되지만, 이 속성을 이용해 캡션의 위치를 아래쪽으로 옮기는게 가능.       
### border 속성 - 표 테두리 스타일 결정하기     
두가지 방법이 있는데, 
```<table border='1'>```
표에 테두리를 그냥 기본값으로 표시.     
다른 방법으로는 CSS의 border 속성으로 표시.     
이때 표의 바깥 테두리와 셀의 테두리를 따로 지정해줘야 함.       
### border-collapse 속성 - 테두리 통합, 분리하기        
```border-collapse: collapse | separate```          
위에서 본 것 처럼 표의 바깥 테두리와 셀 각 테두리가 떨어져 있는데, 
두 테두리를 합칠 것인지 결정하는 속성.      
#### 속성 값        
+ collapse : 테두리를 하나로 합쳐 표시.     
+ **separate** : 테두리를 따로 표시. 기본값.            
### border-spacing 속성 - 인접한 셀 테두리 사이 거리 지정하기       
```border-spacing: <크기>```            
만약 위 속성에서 테두리를 분리 했을 경우,       
인접한 셀 테두리 사이의 거리를 지정.        
#### 속성 값        
+ <크기> : px, em 같은 크기와 단위를 직접 지정.     
속성 값을 하나만 지정한다면 수평, 수직거리를 한꺼번에 지정한 것이고,    
속성 값이 두 개라면 수평거리, 수직거리의 값.        
### empty-cells 속성 -  빈 셀의 표시 여부 지정하기      
```empty-cells: show | hide```          
이것도 테두리 분리된 경우,      
내용이 없는 빈 셀들의 표시 여부를 지정.     
#### 속성 값        
+ **show** : 빈 셀 주위에 테두리를 그려 빈 셀을 표시. 기본 값       
+ hide : 빈 셀에 테두리를 그리지 않고 비워 둠.      
### width, height 속성 -  표의 너비와 높이 지정하기     
표의 너비/높이나 셀의 너비/높이를 지정 할 수 있다.      
만약 표의 너비를 지정하고 셀의 너비를 지정하지 않았다면     
1. 표의 너비를 셀의 개수로 나누어 셀의 너비를 일정하게 지정하고,    
2. 내용이 있다면 내용에 따라 너비가 동일한 비율로 지정.     
### table-layout 속성 -  콘텐츠에 맞게 셀 너비 지정하기     
```table-layout: fixed | auto```            
만약 셀과 표의 너비를 지정했다면, 한글은 괜찮지만        
영문 내용을 여백 없이 길게 입력할 경우, width 속성이 무시되고       
셀 너비가 변하게 됨.           
이때 이 속성을 사용해 셀 안의 내용 양에 따라 셀 너비를 변하게 할지, 고정시킬지 지정.        
#### 속성 값        
+ fixed : 셀 너비를 고정.       
+ **auto** : 셀 내용에 따라 셀 너비가 달라짐. 기본 값.      
근데 **fixed**으로 설정해 너비를 고정하면 셀 너비보다 긴 내용은 셀 밖으로 밀려나가 버림.        
-> word-break:break-all 속성을 추가해야 하고, 셀 height값을 auto로 지정해야 함.     
### text-align 속성 - 셀 안에서 수평 정렬하기       
```text-align: left | right | center```             
셀 안에서 텍스트의 수평 정렬 방법을 지정.       
### vertical-align 속성 - 셀 안에서 수직 정렬하기       
```vertiacl-align: baseline | top | bottom | middle | sub | super | text-top | text-bottom | <길이 값> | <백분율 값>```         
수직 정렬방법을 지정하는 속성. inline이나 inline-block으로 배치한 요소의 세로 정렬 방법으로 지정.       
#### 속성 값        
+ baseline : 인라인 요소의 기준선을 부모요소의 기준선에 맞춘다.     
+ sub : 인라인 요소의 기준선을 부모 요소의 아래 첨자 위치에 맞춤.       
+ super : 인라인 요소의 기준선을 부모 요소의 위 첨자 위치에 맞춤.   
+ top : 인라인 요소의 윗 부분을 부모 요소의 윗 부분에 맞춤.     
+ middle : 인라인 요소의 중앙 부분을 부모 요소의 기준선에서 x-높이(소문자 x의 높이 값)의 반만큼 올려서 맞춘다.      
+ bottom : 인라인 요소의 아랫부분을 부모 요소의 아랫부분에 맞춤.    
+ text-top : 인라인 요소의 윗부분을 부모 요소 글꼴의 윗부분에 맞춤.     
+ text-bottom : 인라인 요소의 아랫부분을 부모 요소 글꼴의 아랫부분에 맞춤.      
+ <길이 값> : 기준선을 0px. 양수면 기준선 위, 음수는 기준선 아래.       
+ <백분율 값> : 기준선을 0%. line-height의 %에 따라 양수면 위, 음수면 아래.     
표의 셀에서 사용될 경우 기준선이나 위, 아래, 가운데 등으로 정렬.    
(실질적으로 표 셀안에서 사용하는 것들)
+ baseline : 셀의 기준선에 내용의 기준선을 맞춤.        
+ top : **패딩**의 위쪽 가장자리에 내용의 윗 부분을 맞춤.       
+ middle : **패딩** 박스 중앙에 내용을 맞춤.        
+ bottom : **패딩**의 아래쪽 가장자리에 내용의 아랫부분을 맞춤.
